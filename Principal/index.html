O SITE SUPREMO

Sobre oque é este site?

Este é um site sobre os meus conhecimentos que obtive durante meus estudos em front-end e back-end. As funcionalidades serão infiltradas em funções para apenas demonstrá-las. Observará versões de mim mesmo e sobre minha personalidade na sociedade.

No que este site vai proporcionar?

Para o mundo, nada. Mas para uma forma de enxergar a evolução de um programador intermediário. Começando com conhecimentos básicos como o HTML e o CSS.



HTML E CSS

Aprendi o HTML no primeiro ano do meu curso técnico em ADS (Análise e Desenvolvimento de Sistemas). O HTML tem um objetivo simples que é construir o site, tanto em base como em uma construção completa. Neste quesito, o CSS se encaixa como um estilizador, trazendo cores, estéticas, animações e entre outros exemplos.
